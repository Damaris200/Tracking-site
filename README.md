# TrackFlow - Modern Shipment Tracking System

![TrackFlow Banner](./public/screenshot.png)

> A production-ready, modern shipment tracking interface built with React, Vite, TypeScript, and Tailwind CSS. Features real-time tracking visualization, comprehensive timeline views, light/dark mode support, and a professional SaaS design.

## âœ¨ Features

### Core Functionality

- **Real-time Shipment Tracking** - Track packages with detailed status updates and location information
- **Interactive Timeline** - Visualize the complete journey of each shipment with timestamps and descriptions
- **Smart Search** - Instant tracking number lookup with loading states and error handling
- **Comprehensive Details** - View origin, destination, carrier, weight, ETA, and delivery progress

### Design & UX

- **Light & Dark Mode** - Seamless theme switching with system preference detection and localStorage persistence
- **Responsive Design** - Pixel-perfect layouts on mobile, tablet, and desktop devices
- **Professional UI** - Modern SaaS design inspired by Stripe, Vercel, Linear, and Supabase
- **8px Spacing System** - Consistent, mathematical spacing scale for visual harmony
- **Smooth Animations** - Polished transitions and hover states throughout

### Technical Excellence

- **TypeScript** - Full type safety with comprehensive interfaces and type definitions
- **Component Architecture** - Reusable, isolated React components following best practices
- **Modern Tooling** - Built with Vite for lightning-fast development and optimized builds
- **Tailwind CSS v4+** - Utility-first styling with dark mode support
- **Performance Optimized** - Efficient rendering, code splitting, and minimal bundle size


### Implementation

- Theme state managed in `ThemeToggle.tsx` component
- CSS custom properties defined in `index.css` with `:root` and `.dark` selectors
- Tailwind's `dark:` variant used throughout components
- Smooth transitions between themes via CSS
